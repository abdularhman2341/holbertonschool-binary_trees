 Binary trees

## Description

A **binary tree** is a hierarchical data structure in which each node has at most two children, referred to as the *left* child and the *right* child. The topmost node is called the **root**, and nodes with no children are called **leaves**. Binary trees are a fundamental building block in computer science and are used in everything from search algorithms and database indexing to expression parsing and priority queues.

This project implements binary trees from scratch in C. It covers the full lifecycle of a tree: creating nodes, inserting children, deleting subtrees, traversing in different orders, and measuring tree properties such as height, depth, size, and balance factor.
